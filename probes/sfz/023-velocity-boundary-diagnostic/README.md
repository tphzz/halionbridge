# 023 Velocity Boundary Diagnostic

This probe checks whether the SFZ velocity split at `63/64` survives conversion
to HALion. It is designed for deterministic comparison between sforzando and
HALion 7 without relying on keyboard playing.

## Files

- `source/001_synth-single-cycle-three-regions-three-velocity_layers.sfz`
  - Reference SFZ for sforzando.
- `source/samples/*.wav`
  - Copied single-cycle sample sources used by the SFZ.
- `halionbridge-build/`
  - Generated HALion Lua build directory.
- `halionbridge-build/001_synth_single_cycle_three_regions_three_velocity_layers.vstpreset`
  - HALion preset generated by halionbridge.
- `plugnscript/halionbridge_sfz_velocity_boundary_generator.cxx`
  - Blue Cat Plug'n Script MIDI generator.

## Manual Validation Loop

1. Load the SFZ from `source/` in sforzando.
2. Load the generated `.vstpreset` from `halionbridge-build/` in HALion 7.
3. Insert Blue Cat Plug'n Script before the sampler under test.
4. Load `plugnscript/halionbridge_sfz_velocity_boundary_generator.cxx`.
5. Start host transport.
6. Compare sforzando and HALion output by ear and, preferably, with a level
   meter or oscilloscope after the sampler.

For strict A/B checks, route the same Plug'n Script MIDI output to one sampler
at a time, record the sampler output, and compare the recorded clips at the
velocity-boundary note positions.

## Expected Sequence

The generator repeats this deterministic sequence:

- Notes: `36`, `51`, `52`, `67`, `68`, `82`
- Velocities per note: `62`, `63`, `64`, `65`
- Note length: `0.55 s`
- Step spacing: `0.85 s`
- Cycle lead-in: `1.0 s`

Expected SFZ behavior:

- Velocities `62` and `63` trigger the lower velocity layer.
- Velocities `64` and `65` trigger the upper velocity layer.
- The layer change should happen between velocity `63` and `64` for every
  tested key, including the key-range boundaries.

The chosen samples have audibly different waveforms between the lower and upper
layers, so a boundary error should be easier to detect than with level-only
velocity tracking.
