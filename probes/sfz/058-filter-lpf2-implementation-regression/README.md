# 058 Filter LPF2 Implementation Regression

This probe follows `057-filter-lpf2-calibrated-validation`.

Probe `057` showed that static LPF2 filter parity is only approximate between
sforzando/sfizz and HALion. The converter now implements the accepted
best-effort mapping: HALion Classic LP12, unchanged cutoff, measured resonance
calibration, and a high-resonance cap at `86`.

## Files

- `source/*.sfz` are same-number sforzando references.
- `source/samples/saw_A3_single_cycle.wav` is copied from probe `057`.
- `plugnscript/halionbridge_sfz_filter_long_generator.cxx` is copied from
  probe `057` and emits one long note at velocity 127.
- `halionbridge-build/*.lua` are generated by `halionbridge convert sfz`.
- `halionbridge-build/*.vstpreset` are built through HALion and ready to load.

## Generate And Build

```powershell
build-release\halionbridge_artefacts\Release\halionbridge.exe convert sfz probes\sfz\058-filter-lpf2-implementation-regression\source probes\sfz\058-filter-lpf2-implementation-regression\halionbridge-build --overwrite
build-release\halionbridge_artefacts\Release\halionbridge.exe probes\sfz\058-filter-lpf2-implementation-regression\halionbridge-build --timeout-seconds 120
```

## Cases

| Case | Source behavior | Expected generated HALion mapping |
| --- | --- | --- |
| `000` | no filter | no filter |
| `001` | `lpf_2p cutoff=1000` | Classic LP12, cutoff `1000`, resonance `32` |
| `002` | `lpf_2p cutoff=1000 resonance=6` | Classic LP12, cutoff `1000`, resonance `48` |
| `003` | `lpf_2p cutoff=1000 resonance=40` | Classic LP12, cutoff `1000`, resonance capped at `86` |

## Measurement Loop

1. Put Blue Cat Plug'n Script immediately before sforzando or HALion.
2. Load `plugnscript/halionbridge_sfz_filter_long_generator.cxx`.
3. Compare same-number pairs from `000` through `003`.

Do not expect tight null-test parity for filtered cases. This probe verifies
that the converter emits the accepted best-effort mapping and that the capped
high-resonance case avoids the rough `90` resonance candidate from `057`.
