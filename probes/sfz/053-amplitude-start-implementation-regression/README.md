# 053 Amplitude Start Implementation Regression

This probe follows `052-amplitude-start-diagnostic`.

Probe `052` showed that sforzando matched the HALion start-level `0`
candidate for `ampeg_start=25`, `50`, and `100`. The converter now emits a
zero initial amp-envelope level for `ampeg_start` cases to match that behavior.

## Files

- `source/*.sfz` are same-number sforzando references copied from probe `052`.
- `source/samples/amp_env_loop.wav` is copied from probe `052`.
- `plugnscript/halionbridge_sfz_amp_envelope_generator.cxx` is copied from
  probe `052`.
- `halionbridge-build/*.lua` are generated by `halionbridge convert sfz`.
- `halionbridge-build/*.vstpreset` are built through HALion and ready to load.

## Generate And Build

```powershell
build-release\halionbridge_artefacts\Release\halionbridge.exe convert sfz probes\sfz\053-amplitude-start-implementation-regression\source probes\sfz\053-amplitude-start-implementation-regression\halionbridge-build --overwrite
build-release\halionbridge_artefacts\Release\halionbridge.exe probes\sfz\053-amplitude-start-implementation-regression\halionbridge-build --timeout-seconds 120
```

## Measurement Loop

1. Put Blue Cat Plug'n Script immediately before sforzando or HALion.
2. Load `plugnscript/halionbridge_sfz_amp_envelope_generator.cxx`.
3. Compare same-number pairs from `000` through `009`.

The generated HALion presets should match the `052` start-level `0` candidates.
