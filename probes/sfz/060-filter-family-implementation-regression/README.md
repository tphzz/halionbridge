# 060 Filter Family Implementation Regression

This probe follows `059-filter-family-rough-mapping`.

Probe `059` selected rough HALion settings for the sforzando-recognized static
SFZ filter families. This probe uses the same source set, but the HALion Lua is
generated by `halionbridge convert sfz`, so it validates the converter
implementation rather than hand-authored candidates.

## Files

- `source/*.sfz` are same-number sforzando references copied from probe `059`.
- `source/samples/saw_A3_single_cycle.wav` is copied from probe `059`.
- `plugnscript/halionbridge_sfz_filter_long_generator.cxx` is copied from
  probe `059` and emits one long note at velocity 127.
- `halionbridge-build/*.lua` are generated by `halionbridge convert sfz`.
- `halionbridge-build/*.vstpreset` are built through HALion and ready to load.

## Generate And Build

```powershell
build-release\halionbridge_artefacts\Release\halionbridge.exe convert sfz probes\sfz\060-filter-family-implementation-regression\source probes\sfz\060-filter-family-implementation-regression\halionbridge-build --overwrite
build-release\halionbridge_artefacts\Release\halionbridge.exe probes\sfz\060-filter-family-implementation-regression\halionbridge-build --timeout-seconds 120
```

## Measurement Loop

1. Put Blue Cat Plug'n Script immediately before sforzando or HALion.
2. Load `plugnscript/halionbridge_sfz_filter_long_generator.cxx`.
3. Compare same-number pairs from `000` through `020`.

Cases `013` through `016` and `020` intentionally emit unsupported-filter
warnings and no HALion filter block. Their presets are still built so the
source set remains same-number and easy to inspect.

## Result

Generated on 2026-06-14. Converter warnings were emitted for `_sv` cases
`013` through `016` and `pink` case `020`.

Generated Lua inspection confirmed:

- `001_lpf_1p`: LP6, cutoff `700`, resonance `0`.
- `004_lpf_6p`: dual serial, ShapeA LP24, ShapeB LP12, resonance `59`.
- `011_brf_1p`: BR12, resonance `35`, preserving authored `brf_1p` instead of
  sfizz's resolved `brf_2p`.
- `020_pink`: no filter block.

HALion build completed successfully on 2026-06-14: 21 scripts processed,
21 `.vstpreset` files saved.
